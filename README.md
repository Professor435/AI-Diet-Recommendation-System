# AI-Diet-Recommendation-System
AI-Diet-Recommendation-System
This NutriGenius AI system features:
🎨 Design Highlights
Glassmorphism UI with animated gradient backgrounds and morphing blobs
Neumorphic buttons with depth effects and hover animations
Floating animations and micro-interactions throughout
Space Grotesk + Inter font combination for modern typography
Dark theme with cyan (#00D9C0), coral (#FF6B6B), and yellow (#FFE66D) accents
⚡ Fully Functional Features
1. Interactive Dashboard
Real-time calorie tracking with animated progress bars
Water intake tracker with visual cup indicators (click to add water)
Dynamic macro charts using Chart.js
Weekly trend analysis with switchable views (Calories/Macros/Weight)
2. AI Food Analyzer (3 Input Methods)
Text Analysis: Type meal descriptions, get instant nutrition breakdown
Voice Input: Simulated speech-to-text with recording animation
Image Upload: Photo analysis simulation with preview
Smart food detection with confidence scores
AI recommendations based on meal composition
3. Smart Meal Planner
Diet type selection (Keto, Vegan, Paleo, Mediterranean)
Calorie target slider with live updates
AI-generated meal cards with nutrition info
Interactive shopping list with check-off functionality
4. Progress Tracking
Weight progression charts
Achievement badges with streak counters
Animated circular progress rings
Detailed analytics cards
5. Quick Calculator Modal
BMR/TDEE calculator with activity multipliers
BMI calculation with health categorization
Weight loss/gain calorie recommendations
6. Profile Management
Slide-out profile panel
Editable user stats
Goal and activity level settings
🚀 Interactive Elements
All buttons are functional (no placeholders)
Real-time data updates and animations
Toast notifications for user actions
Smooth section transitions
Responsive design for all screen sizes
Hover effects and micro-interactions on every element
The system uses Chart.js for data visualization, Tailwind CSS for styling, and vanilla JavaScript for all functionality. Every feature works end-to-end with realistic data simulation!
